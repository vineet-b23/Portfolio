Task Reminder App

A Django-based web application designed to help users manage tasks efficiently with automated reminders. Built during my time at Wiztech Automation Solutions , this project focuses on backend automation, clean UI, and real-world usability.

What it does

Users can create, update, and manage tasks through a clean interface, while reminders are automatically triggered using Django management commands and the Windows Task Scheduler.


Key Highlights

  • Full CRUD functionality for task management
  • Automated reminders without third-party services
  • Bootstrap-based responsive UI
  • Clean project structure with proper documentation
  • Real-world automation workflow testing & debugging

This project strengthened my understanding of backend automation, debugging scheduled jobs, and writing production-ready Django applications.
